# Build from source
## Linux

```bash
sudo apt install cmake build-essential libsdl2-dev libsdl2-gfx-dev libsdl2-image-dev libvulkan-dev vulkan-validationlayers libglm-dev libglew-dev libgl-dev libglew-dev
mkdir build
cmake .. -GNinja
ninja
```


### Cross-Compile - Windows
- sudo apt install mingw-w64 binutils-mingw-w64
